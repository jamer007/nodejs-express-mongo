# nodejs-ms-campaign

## Getting Started

Install dependencies:
```sh
npm install
```

Setup GitHooks:
```sh
npm run setup-git-hooks
```

Start server:
```sh
# Start server
npm start
```

Lint:
```sh
# Check code lint
npm run lint

# Lint code with ESLint
npm run lint --fix
```

Docs:
```
# access REST API documentation
http://localhost:4410/api-docs/
```
